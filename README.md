# Public Key Cryptography course notes

## For Eötvös Loránd University Budapest, Cyber Security specialisation, spring semester 2026.

This repository contains LaTeX files and compiled PDFs for the above course.
There are no slides for the lectures, so I felt it necessary to save the course
materials digitally too in addition to my notebook. I welcome any and all
contributions, especially since I cannot take note of everything that has been
said on the lectures on my own.

The recommended LaTeX distribution for compiling the LaTeX files is [TeX
Live](https://www.tug.org/texlive/).
